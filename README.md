# LLM Document Links

A curated and structured collection of documentation, technical references, and best‑practice resources related to Large Language Models (LLMs).  
This repository serves as a **personal knowledge base** to quickly access essential information for prompting, APIs, cloud platforms, model capabilities, and evaluation techniques.

---

## 📚 Purpose

This repository is designed to:

- Organize LLM‑related documentation in a clean, searchable structure  
- Provide quick access to official references  
- Capture personal notes and key takeaways for future learning  
- Support continuous study for AI, cloud, and machine learning certifications  
- Serve as a long‑term knowledge hub for LLM development

---


Each file includes:

- A link to the official documentation  
- A short summary  
- Key takeaways  
- Space for personal notes  

---

## 📌 Categories Overview

### **Prompt Engineering**
Guides and best practices for designing effective prompts across different LLMs.

### **API Documentation**
Official references for Gemini, OpenAI, Claude, and other model APIs.

### **Cloud Platforms**
Documentation for AWS Bedrock, Azure OpenAI, and Google Vertex AI.

### **Models**
High‑level overviews of major LLM families: Gemini, Claude, GPT, Llama.

### **Evaluation**
Resources for RAG evaluation, LLM benchmarks, and performance assessment.
